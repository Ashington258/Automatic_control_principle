对于给定的开环传递函数 \( G(s) \) 和单位负反馈系统，闭环传递函数 \( T(s) \) 可以通过以下公式计算：

\[
T(s) = \frac{G(s)}{1 + G(s)}
\]

将之前的开环传递函数 \( G(s) = \frac{K_1(s + 1)}{s^2(s + a)} \) 代入，得到：

\[
T(s) = \frac{\frac{K_1(s + 1)}{s^2(s + a)}}{1 + \frac{K_1(s + 1)}{s^2(s + a)}}
\]

进一步简化：

\[
T(s) = \frac{K_1(s + 1)}{s^2(s + a) + K_1(s + 1)}
\]

这就是单位负反馈系统的闭环传递函数。

\[
{s^2(s + a) + K_1(s + 1)}=0
\]
